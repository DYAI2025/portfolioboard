# Bilder-Ordner

Lege deine Bilder hier ab. Nach dem Build/Deploy sind sie unter `/images/filename.jpg` erreichbar.

## Verwendung im Editor

Gib im Tile-Editor bei **Image URL** einfach den relativen Pfad ein:

```
/images/dein-bild.jpg
```

## Beispiele

- `/images/projekt1.jpg`
- `/images/portfolio.png`
- `/images/background.webp`
